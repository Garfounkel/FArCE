# How to compile the project:

- Go into 'src' folder.
cmd:	cd src

- Compile the project (you need the SDL libraries).
cmd: 	make


# How to use it:

- Go into 'src' folder.
cmd:	cd src

- Execute the binary with an image for parameter (all images type working with SDL should be working).
cmd: 	./main [yourImage]

Exemple: ./main ../database/adjou_k
